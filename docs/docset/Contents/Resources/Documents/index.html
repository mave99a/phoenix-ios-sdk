    TSPhoenix Reference

    Overview

    TSPhoenix is an iOS SDK for Tigerspike Phoenix Platform. By using the SDK instead of the HTTP APIs, you can save massive amount of development time, interface with high-level APIs and focus your energy on the UI.

    • Native Objective-C APIs
    • Seamless integration with Cocoapods
    • User management: log in, out, auth tokens, etc.
    • Transparent caching via Core Data
    • High level models such as Articles and Sections, instead of JSON dictionaries and arrays
    • Maintained by Tigerspike Products team, so you don’t have to
    • Built on solid open-source components you know and love: AFNetworking and MagicalRecord.
    • API response pagination is taken care of
    • Flexible and extensible
    • Fully documented, Appledoc style
    • and much more

    Modules

    TSPhoenixClient

    A singleton entry to all SDK modules. It is a standard AFHTTPClient subclass which handles all the underlying network traffic. [TSPhoenixClient sharedInstance]

    all modules are a property of the shared instance, i.e. [TSPhoenixClient sharedInstance].identity [TSPhoenixClient sharedInstance].syndicate

    TSPhoenixIdentity

    Phoenix Identity module, responsible for OAuth logging in / out, loading memberships, etc.

    TSPhoenixSyndicate

    Phoenix Syndicate module, responsible for loading and caching sections / articles, sending article interactions back to Phoenix.

    Example usage

    Logging in

    TSPhoenixClient *client = [TSPhoenixClient sharedInstance];

[client.identity authenticateWithUsername:self.userField.text
                                 password:self.passwordField.text
                                  success:^(AFOAuthCredential *credential) {} failure:^(NSError *error) {}];

    Syndicate: list all subscribed sections

        [[TSPhoenixClient sharedInstance].syndicate loadSubscribedSectionsWithSuccess:^(TSPaginator *paginator, NSArray *objects, NSUInteger page) {
            // Populate the UI
    } failure:nil];

    Syndicate: list subscribed articles

    [[TSPhoenixClient sharedInstance].syndicate loadArticlesInSection:self.section
                               fromDate:fromDate
                                 toDate:[NSDate date]
                            success:^(TSPaginator *paginator, NSArray *objects, NSUInteger page) {} failure:^(TSPaginator *paginator, NSError *error) {}];
